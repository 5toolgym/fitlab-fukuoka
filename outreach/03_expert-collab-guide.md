# 専門家コラボ アプローチガイド

「福岡の専門家」カテゴリの対談記事を制作するための
アプローチ手順・テンプレート・進捗管理表。

---

## アプローチの流れ

```
Instagram DM で第一接触
    ↓
関心を示してもらえたらメールで詳細を説明
    ↓
ZoomまたはLINE通話で30〜45分の取材
    ↓
記事案を送ってレビューしてもらう
    ↓
公開・SNSで相互シェア
    ↓
公開後2〜3週間でフォローアップ（効果共有）
```

---

## Instagram DM テンプレート

**用途**: 最初の接触。短く・メリットを明確に。

```
（宛名）さん、はじめまして。
福岡でパーソナルジム「5toolgym」を運営している柴山といいます。

（相手の専門分野）と姿勢改善・カラダの機能改善は深く関連していると感じており、
ぜひ一度お話を聞かせていただきたいと思いメッセージしました。

弊社が運営する情報メディア「FitLab Fukuoka」
（https://fitlab-fukuoka.pages.dev）で
「（相手の専門分野）×パーソナルジム」というテーマで
取材記事を制作させていただきたいと考えています。

記事は（相手のURL等があれば）先生のご紹介・プロフィールリンクも掲載いたします。
ご関心あれば、もう少し詳しくお伝えできればと思います。

よろしくお願いいたします。
5toolgym 柴山智幸
```

## メール依頼文テンプレート

**用途**: DM後に関心を持ってもらえた方へ詳細を送る。

### 件名
```
FitLab Fukuoka 取材記事の件（5toolgym 柴山より）
```

### 本文
```
（宛名）先生

先日 Instagram にてご連絡した、5toolgym の柴山智幸と申します。
ご返信いただきありがとうございます。

改めてご提案の概要をお伝えいたします。

■ FitLab Fukuoka について
弊社が運営するパーソナルトレーニング情報メディアです。
URL: https://fitlab-fukuoka.pages.dev
福岡エリアで姿勢改善・ダイエット・ジム選びを検討している方々に
向けた記事を定期発信しています。

■ 今回の依頼内容
「（テーマ）」をテーマに、先生の専門家視点からのコメント・お考えを
取材させていただき、記事として構成したいと考えています。

【記事の流れ（イメージ）】
1. 冒頭: テーマの背景説明（柴山が執筆）
2. 中盤: 先生との対談形式（Q&A or 発言引用）
3. まとめ: 読者へのアドバイス

【先生のメリット】
・記事内でプロフィール・ご連絡先（インスタ等）を紹介
・公開後、5toolgymのSNS（Instagram 3アカウント）でシェア
・被リンクとして SEO にも有効

【取材方法】
ZoomまたはLINE通話（30〜45分）、もしくはメールでのやりとりでも構いません。

■ ご確認いただきたいこと
1. 取材ご協力いただけそうでしょうか？
2. 都合のよい日時・連絡方法を教えていただけますか？

記事の草稿は公開前に必ず先生にご確認いただきます。
内容に問題があれば修正・差し戻しも可能ですのでご安心ください。

ご検討いただけますと幸いです。
よろしくお願いいたします。

─────────────────────
5toolgym オーナー 柴山智幸
Email: 5toolgym@gmail.com
FitLab Fukuoka: https://fitlab-fukuoka.pages.dev
─────────────────────
```

## アプローチ先の探し方

### Instagramの検索キーワード（福岡の専門家を探す）

| 専門家の種類 | 検索キーワード（Instagram） |
|------------|--------------------------|
| 整骨院・整体院 | #福岡整骨院 #福岡整体 #中央区整骨院 |
| 管理栄養士 | #福岡管理栄養士 #福岡ダイエット管理栄養士 |
| ヨガインストラクター | #福岡ヨガ #福岡ヨガインストラクター |
| 理学療法士（PT） | #福岡理学療法士 #福岡PT |
| 美容・エステ | #福岡エステ #福岡美容 #中央区エステ |
| 医師（内科・整形外科） | #福岡医師 #福岡整形外科 |

### 良い候補者の見分け方
- フォロワー数1,000〜10,000程度（多すぎると返信来ない）
- 専門的な情報発信をしている（「ただのお知らせ」だけでない）
- 最近も更新している（3ヶ月以内の投稿あり）
- プロフィールに連絡先（LINE・メール）がある

## 記事公開後のフォロー手順

公開後2〜3日以内に行う。

1. **先生に公開URLを送る**
   ```
   （宛名）先生
   先日お話を伺った記事が公開になりました。
   ご確認ください: （記事URL）

   ご協力いただきありがとうございました。
   もし内容に気になる点があればいつでもご連絡ください。
   ```

2. **5toolgymのInstagram 3アカウントでストーリーズシェア**
   - 記事へのリンクをストーリーズに貼る
   - 先生のアカウントをタグ付けする

3. **先生にSNSシェアをお願いする（任意）**
   ```
   もしよろしければ、先生のSNSでも記事をシェアいただけますと大変嬉しいです。
   （任意ですので、ご都合に合わせてください）
   ```

## よくある断られ方と対処法

| 断り文句 | 対処法 |
|---------|--------|
| 「時間がない」 | 「メールでのやりとりのみでも構いません。質問5問をメールで送りますので、お時間のあるときにご回答いただくだけで大丈夫です」 |
| 「顔出しNG」 | 「お顔の写真は不要です。アイコン表示・イラストでの紹介も可能です」 |
| 「どんな記事になるか不安」 | 「公開前に必ず草稿をご確認いただきます。内容に問題があれば修正・公開中止も可能です」 |
| 「ライバル店に利用されそう」 | 「記事の目的は先生の専門知識を読者に届けることであり、競合との比較はしません。先生のお名前・連絡先を紹介することが主な目的です」 |
