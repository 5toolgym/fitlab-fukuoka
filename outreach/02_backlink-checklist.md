## 5. Yahoo!ロコ・Yelp Japan 登録手順

### Yahoo!ロコ
- [ ] https://loco.yahoo.co.jp/ にアクセス
- [ ] 「店舗オーナーの方はこちら」から登録
- [ ] 薬院店・赤坂店それぞれで店舗情報を登録
- [ ] ウェブサイト欄に FitLab Fukuoka の URL も記載

### Yelp Japan
- [ ] https://biz.yelp.co.jp/ にアクセス
- [ ] 「ビジネスをクレーム」から両店舗を登録・更新
- [ ] ウェブサイト欄に FitLab Fukuoka の URL を追加

**メモ**: これらのディレクトリへの登録は SEO のドメイン権威（Domain Authority）を高める効果がある。設定後は定期的に口コミの確認・返信を行う。

## 6. プレスリリース配信テンプレート

FitLab Fukuoka 公開時や記念記事公開時に使えるプレスリリース文。
PR TIMES や ValuePress で配信することで被リンクが得られる。

### テンプレート
```
【プレスリリース】

タイトル:
5toolgym（福岡）、パーソナルトレーニング情報メディア
「FitLab Fukuoka」を開設——姿勢改善・ダイエットの専門情報を発信

本文:
福岡市中央区でパーソナルトレーニングジム「5toolgym」（薬院店・赤坂店）
を運営する柴山智幸（NCCA講師）は、オウンドメディア「FitLab Fukuoka」
（https://fitlab-fukuoka.pages.dev）を[開設日]に公開いたしました。

本メディアでは、体性感覚トレーニング・機能改善指導・姿勢改善・
福岡のパーソナルジム選びなどに関する記事を定期的に発信します。
福岡エリアで運動・健康改善を検討している方々に向けた実践的な情報提供を
目的としています。

■ FitLab Fukuoka 概要
URL: https://fitlab-fukuoka.pages.dev
運営: 5toolgym（福岡市中央区）
執筆者: 柴山智幸（NCCA講師・体性感覚トレーニング専門）
カテゴリ: 姿勢改善 / ダイエット / トレーニング / ジム選び / 栄養・食事 / 福岡の専門家

■ 5toolgym について
薬院店: 福岡市中央区薬院3丁目13-22 新宝ビル3F（薬院大通駅徒歩2分）
赤坂店: 福岡市中央区舞鶴3丁目2-7 K2ビル1階（赤坂駅徒歩6分）
```

- [ ] PR TIMES または ValuePress に登録済み
- [ ] FitLab 公開から2週間以内に配信する
- [ ] 配信後のリンク被リンクを確認（Google Search Console）
